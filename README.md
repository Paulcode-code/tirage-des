# Tirage 2 dés

Petite webapp avec 3 modes :

1. Mode 1 : vrais dés aléatoires.
2. Mode 2 : paquet de 36 tirages, comme une pioche.
3. Mode 3 : probabilités qui se rééquilibrent après chaque tirage.

## Lancer en local

Ouvre simplement `index.html` dans ton navigateur.

## Déployer sur Vercel

Importe ce dossier dans un dépôt GitHub, puis connecte ce dépôt à Vercel.
